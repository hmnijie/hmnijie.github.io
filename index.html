为什么有的男的奇丑无比，却把到了无数妹子？

发表于 2020-04-10 分类于 转载

我们首先要明白一个心理学、神经科学的事实，那就是人的生物性。

气场是确确实实存在的。不同的人，有不同的气场。

简单来说，一个具体的人，他的各种荷尔蒙激素，通过身体散发，对方接受后，产生反应。这个跟他（她）的长相、穿着等都无关。人就是被动的接受了这些化学物质，然后有了反应。这种反应可能是兴奋、可能是更愿意与对方交往，也可能是沮丧，无助等。

所以日常生活中一些外表很不怎样，甚至口才、举止等也很不怎样的人，男的或者女的，就是让人有亲和感，甚至就是让人愿意进一步亲密，为什么？

没有味道，却确确实实存在的荷尔蒙存在。

化学物质对人的影响是非常大的，烟、酒、药物，全世界化那么大力气控制，就能明白它们的厉害。大部分的修炼、宗教，根本上来说，也就是通过一定的方式，刺激身体产生某种特定激素，让人平和、喜乐；或积极、奋进。

还有，把妹，把妹，靠的是把，把长妹多，与长相无关。

内不欺己 外不欺人

发表于 2016-11-28 分类于 琐碎

每天都在纠结，所以很累。怎么形容我的纠结呢？像是时不时被女友询问可不可以去逛商场的无奈男友，像是一天到晚调皮捣蛋的熊孩子的妈 ，像是鼻子上被放着一摞又一摞饼干的金毛。现在发现了，是我嘴巴大，不老实，爱骗人，但三观又正，总是对自己做彻底的自我批评，这样折腾自己当然累，简直有病。

现状

发表于 2016-11-24 更新于 2016-11-25 分类于 琐碎

从上周起，开始有意识地离开网络，将上网的时间用来看一些书。
今天我在电脑前坐了一下午，浪费了许多时间去浏览垃圾、色情的信息，不应该的。我很难将万维网视为一种工具，毕竟那些服务器里由1、0组成的巨量数据曾满足了我太多的心理、生理需求。甚至，是网络维系了我目前的社交关系，让我不至于消失于人前。一百多年前，如果有一个人不接受电灯的普及，现在看来，那个人愚昧的，是要淘汰于历史的潮流之中的。网络最终会改变这个社会，信息的洪流会冲刷我们这代人，并颠覆封闭性的旧社交方式，改造会经历足够长的历史跨度。我不想当那个抗拒电灯泡的人，但我对现有的网络形式充满着深深的疑虑，毕竟网络在中国的兴起也就是近二十年的事情，其对我的成长真的是正面的吗？也许二十一世纪初的信息化是人类第一次全球性质的大实验，我们都是小白鼠。

现状是我离不开网络，但网络是一种的工具，只配充当我生活的极小一部分。

焦躁 舒适

发表于 2016-10-21 更新于 2016-10-24 分类于 琐碎

夏季的湿热延续至今，云层遮蔽住阳光，使大地的水分不断凝聚进空气。珍珠（台风）将至，大概又是几天的雨。

二十五度以上的气温，八十以上的相对湿度，汗水已不能蒸发散热。

焦躁！焦躁！

此时，好想有一间房间，房间是23°恒温的，湿度适宜，明亮干净，安着柔软的大床。我和爱人赤身裸体地躲进被子，她趴在我的身上，我双手抚摸她的腰臀。拥抱、亲吻…… 酣悦后，闻着熏香，听着音乐，轻道晚安。一切都是那样舒适美好。

我离“舒适”还有多远呢？

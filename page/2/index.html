南京

发表于 2016-10-03 更新于 2020-03-24 分类于 琐碎

稍稍了解近代史的人，会对南京绝对的敬畏。此地是共和国的诞生地啊，如今落魄到和杭州、苏州比了。

我并不适应南京。在新街口买的加奶美式酸涩得不能喝。上海商场卫生间一般都配上一次性马桶垫了，南京德基广场的卫生间却还没纸用。冲动的老司机们骂来骂去也就一句——“你个呆逼”。

南京的高校密度是仅次于上海的。就宁城一地有实力的高校就比浙一省要多了。苏州前些年的GDP几倍于南京，如今苏州再无苏州人（失去本地文化）。
得益于房产市场，今年有不少本地市民实现资本翻翻。只有等到中国真正是在用知识和文化创造价值时，南京会才能再次成为南“京”吧。毕竟，随便捡块南京的城墙砖，论历史，就要比美国长了。

大脑

发表于 2016-09-11 更新于 2016-09-14

大脑要发育到二十五周岁才算基本成熟。一个人完成本科教育，毕业后再过个两三年也差不多就是二十五。
大脑对激素分泌的控制力不断加强，同时对行为的控制能力会逐渐提高，这才是人从冲动到稳健，从幼稚到成熟的关键因素。
我的行为在价值观的引导下尚算可控，但我决然不能保证能消灭自己贪婪自私的动物性，只有神佛是没有动物性的，而现代中国人恰恰缺乏宗教（宗族）的熏陶。
大脑坏了，人的控制力就坏了。喜欢吃甜品的女孩子，很难再控制糖分的摄入。吸食过一次海洛因的人无疑会吸食第二次。自闭症、抑郁症，厌食症等精神疾病的基本表现就是自控能力的丧失。
我能做的是尽力引导大脑。糖分让人上瘾，所以我尽量不吃甜食。大众媒体上的种种性暗示，非礼勿视，眼不见为净。保证食物干净，少吃洋快餐，激素太多了。常常补充点维生素D，脑子能长好一些的。另外，分清楚对错，少为大脑想犯的错找理由（这是最难的）。
成为一个成熟的男人好难啊！
最后，鄙视王宝强，更鄙视看着庸俗新闻的我。

写在二十岁

发表于 2016-07-11 更新于 2016-07-12

两三个小时前我完成了第三学期的课程设计，以很凑活的方式。不断画图改图，一个礼拜昼夜不停，CAD已经可以轻松使用了。越发觉得这些工具性的软件不需要学，用大量的时间去钻研一个软件，若非赖以为生，则纯属浪费。我终究不想成为一个画图纸的人。
少年得意如芳草，日日春风好，我已没心没肺地度过了大半个青春。半年彷徨，愈发想去实现自己的梦想，并愿意为其经历苦难，付出情感。
一步一步慢慢来，首先得在五年到十年内出国。
我们的征途可是星辰大海哟！

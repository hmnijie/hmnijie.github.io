向右转

发表于 2016-06-24 更新于 2020-03-24

昨晚十二点，欧元涨得很欢，好想买一百美金日元，如果英国人脱欧撞墙，大不了赔去一半，但成功了，下个月的生活费也有了，可我现在连一百美金都拿不出来，太穷了。
两个小时前英国人民选择选择脱欧，很有趣。看昨天选前的最后民调，或者看博彩赔率，英脱欧似乎成为不可能。在很多问题上，英国人和德法是脱节的，德国人经历过被分割的痛苦，法国是欧洲左派思想的输出地，德法都对欧洲的统一与强大有着天然的向往。难民问题爆发后，出于人道，欧洲接受了大量大量的难民，这让我感受到人类终究是在一点点美好起来的，随之而然的是德国汉堡的大范围性侵案和法国巴黎的恐怖袭击，各国的激进右派势力极快地拥有了话语权。美国这个孤悬海外的超级大国也同样右转，把同性恋群体和少数族裔当作衣食父母的希拉里婆婆本当选无疑，如今一个小丑人物却在美国选战中风光无限。
德法英美的底层人民是有选票的，当群众的利益受到了普遍的损失，人民会先逮住最显眼的替罪羊。精明的英国人此时的心理很好猜：“老子工资又少了，都怪那些来英国偷工作的穷老鼠，卡梅伦你不是人！”（正写着，新闻推送说卡梅伦辞职了，他好惨，我好可怜他。）

哼，西方人真矫情，中国房价这么高，我们都没造反呢。对哦，我们连选票都没有，有没有造反但不流血牺牲的办法啊，人家香港学生实在拼，学不来。

差异

发表于 2016-06-16 更新于 2016-06-17

世界上不可能有比中国更多元化的国家了，四十多年前的动乱，国人被强迫改造，精神狂热代替了理性逻辑，三十多年前的政权轮替，实用主义代替共产信仰。如今，中国仍然不断被撕裂着，这不仅仅是因为所谓贫富悬殊。当我们在不断讨论中国人的道德底线时，有人会辩证唯物地说：经济基础决定上层建筑，大部分人的贫穷是低素质的根源。然而上海，北京的土著，都很有钱吧？比比素质？上海是西化了百年的咖啡文化，北京是传承千年的大蒜文化，不一样的。

我们是似乎开放的一代，有人奉行“喜欢就上，没感觉就分开”的恋爱准则，也有人标榜着“不以结婚为目的的谈恋爱的就是帅流氓”的变态逻辑。

女性的自我矮化是这个年代的常态，有女生早早表露了相夫教子的意愿，也有女人不断追求强大，期望能够证明自我价值。中国女权运动的悲剧是——女人好像理所应当地越来越承担起独立自强的责任，然而传统的，封建的生育枷锁依然紧紧地禁锢着女性。

西方七八十年代垮掉得那批人现在也是爷爷奶奶的年纪了，那些老人可以毫无隔阂得和年轻人说着自己年轻时候的故事。中国什么时候能有稳定的、主流的价值观呢？

男人

发表于 2016-06-08

他应该是个男人，简单的说让人感觉到男性的男人。他不必很帅，但是应该长的端正，应该看起来很正直。他应该善良，但是不应该是２４小时微笑的让你发腻。他的血管里要有野兽的血液，会在你意想不到的那一刻，抱着你，吻你；他应该会发脾气，虽然未必伤害你。他应该风在发端，身上的烟草味和酒味让你觉得兴奋。他会读书，他会思考，他会用冷峻的眼神看你的心。
我喜欢的男人一定要坚强，他会哭，但是他不会逃避，他会埋怨，但是不会成天怨天尤人。他在最贫穷的情况下也能快乐的自足，在富裕的情况下也不会四处炫耀。他爱的应该不只是我，而是这个世界。他是个男人，野性，睿智，坚强，爱我的男人。
——李旻

胡老师

发表于 2016-06-04

“你像一个住在宫殿里的、孤独的小孩子。”胡老师在心理辅导里这样形容我，这句话一直觉得不恰当，但确实有意思。
胡建兵是学校现在唯一的主管学生心理的老师，也是心理协会的指导老师，个子矮矮，其貌不扬，精神世界却很神奇。有次不知道他哪里找来一批封建余孽，在学校办讲座，用礼义孝悌给女学生们洗脑了一天，包午饭，管零食。心理辅导上提到算命，他叫我们别不信，真的玄的。不定期花几千几万的钱去参加心理培训，说是去升华心灵。有学生找他心理咨询，有时候他还晾着学生，叫她们改天，然后继续在网络上和人升华心灵。
昨晚，给他灌酒，他说不喝，老婆管得严。最后散伙的时候满脸心事地走了，说得赶快回家交差。哦，他老婆，打麻将认识的。
真是一个人，一个活法。

形式主义

发表于 2016-05-31 更新于 2020-03-24

换过寝室，从一期到二期。
大一是豪华的套间房，现在已在直廊宿舍住满一年了。
入校时有好好动心思打扮寝室，空间很小很小，但装修起来还是很麻烦。后来学校把我们强搬去21幢，并雇人重新粉刷了我们住过的房间。等我再去看我的床位的时候，那些墙纸啊，木架子啊都已经没有了，很心痛的。后来住进的女生不会讨厌的吧？何必。
以后再也不搞了，难看，就难看着吧，搬了寝室后我是这样想的。
形式主义很让我烦恼，如果手上没一只好笔，桌上没一杯咖啡，并没一个让我舒适的环境，我几乎无法作业思考。我的生活有太多的多余，太多纷繁杂碎的细节在我的大脑里闪现，太累了。
这或许是一种贪欲，有了一件好衬衫，想着得有一个好衣柜，最后有一套大房子才是最需要的了。
